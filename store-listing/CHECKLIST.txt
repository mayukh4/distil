=== WORKFLOW ===

For each graphic asset:
1. Take a real screenshot of the relevant Distil feature
   (Use Chrome DevTools device mode for exact 1280x800 viewport)
2. Open Gemini (gemini.google.com) — uses Nano Banana 2 by default
3. Upload the real screenshot(s) as reference using the + button
4. Paste the text prompt from the corresponding prompt file
5. If refinement needed, upload the generated image back and ask for edits
6. Verify exact pixel dimensions before uploading to Chrome Web Store
7. Save final assets into store-listing/assets/ directory

=== TIPS ===

- For the icon: existing icon.png may already work — resize/clean up if needed
- For screenshots: if AI-generated versions have font/layout issues, use actual
  browser screenshots instead (1280x800 viewport via DevTools)
- Chrome Web Store rejects assets that don't match exact pixel requirements
- Always attach real screenshots as reference when using image generation
